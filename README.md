# uk-property-analytics

UK housing analytics built on HM Land Registry Price Paid data (~30M rows, 1995→present).
Stack: **dbt-core 1.8** + **DuckDB** + **Streamlit** + **GitHub Actions** + **GitHub Pages** for dbt docs.
Sources → staging → intermediate → marts (dimensions / facts / reporting), tested at every layer.

> Two-week sprint following [`PROJECT-2-KIT.md`](./PROJECT-2-KIT.md). Currently in **Phase 0 — bootstrap**.

The full README (architecture diagram, business questions answered, live links, run instructions) lands in Phase 5.
